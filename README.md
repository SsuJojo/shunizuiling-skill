# shunizuiling-apis

这是一个 OpenClaw Skill，用于把数你最灵（Smartestu）相关能力封装为可调用技能，支持课程、作业与题目查询等场景。

## 安装

非常简单，你只需将以下消息发送给 OpenClaw：

```markdown
# 请安装数你最灵 skill
1. 从 `https://github.com/SsuJojo/shunizuiling-skill.git` 克隆仓库到工作目录的 `skills/` 并更名为 `shunizuiling-apis`
2. 将其作为 **skill** 加载
3. 执行 **Initial implementation scope** 中的指令并标记为已配置
```
